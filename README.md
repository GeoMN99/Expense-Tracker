# 💰 Expense Tracker

A sleek, dark-themed expense tracking web application built with vanilla HTML, CSS, and JavaScript. Set a monthly budget, log your expenses, visualize your spending, and export your data — all stored locally in your browser.

## ✨ Features

- 💵 Set a monthly budget in KES
- ➕ Add, edit, and delete expenses
- 🗂 Categorize expenses (Food, Transport, Entertainment, Shopping, Health, Bills, Other)
- 🔍 Search and filter expenses by name or category
- 📊 Doughnut chart showing spending breakdown by category
- 📈 Budget progress bar that turns amber at 70% and red at 90%
- 💾 All data persists using localStorage — no account needed
- 📥 Export expenses as a CSV file for Excel or Google Sheets
- 📱 Fully responsive — optimized for mobile and desktop

## 🚀 Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/geomn99/Expense-Tracker.git
   ```

2. Navigate into the project folder:
   ```bash
   cd Expense-Tracker
   ```

3. Open `index.html` in your browser — no API key or setup needed!

## 💡 How to Use

1. Enter your monthly budget and click **Set Budget**
2. Fill in the expense form — name, amount, category, and date
3. Click **Add Expense** to log it
4. Use the search bar or category filter to find specific expenses
5. Click **✏️ Edit** to update an expense or **🗑 Delete** to remove it
6. Click **📥 Export CSV** to download your expenses as a spreadsheet
7. Check the doughnut chart to see where your money is going
